# Data Retention Policy

## 1. Purpose

This policy outlines how Fathuss handles data retention for user-submitted code, execution traces, and related platform data.

## 2. Data Categories

### 2.1 User-Generated Code
- Source code submitted for challenges
- Test cases and solutions
- Code execution results

### 2.2 Execution Traces
- Runtime logs and performance metrics
- Error messages and debugging information
- Resource usage statistics

### 2.3 User Data
- Account information
- Challenge participation records
- Leaderboard data

## 3. Retention Periods

### 3.1 Active User Data
- User accounts: Retained indefinitely unless deleted by user
- Challenge submissions: Retained for 2 years after submission
- Execution traces: Retained for 90 days

### 3.2 Inactive Accounts
- Accounts inactive for 2 years: Data archived
- Accounts inactive for 5 years: Data deleted

### 3.3 Legal Holds
- Data subject to legal holds: Retained until hold is released
- Data required for ongoing investigations: Retained as needed

## 4. Data Storage and Security

### 4.1 Storage Locations
- Primary: Encrypted cloud storage
- Backups: Encrypted offsite backups
- Logs: Secure logging systems

### 4.2 Security Measures
- End-to-end encryption for sensitive data
- Access controls and audit logging
- Regular security assessments

## 5. Data Deletion Procedures

### 5.1 User-Initiated Deletion
- Users can request account deletion through settings
- All associated data deleted within 30 days
- Some anonymized data may be retained for analytics

### 5.2 Automatic Deletion
- Expired data automatically deleted per retention schedule
- Secure deletion methods used (cryptographic erasure)

## 6. Data Minimization

We collect and retain only data necessary for platform functionality and legal compliance.

## 7. Audit and Compliance

- Regular audits of data retention practices
- Compliance with GDPR and other privacy regulations
- Data retention logs maintained for accountability

## 8. Policy Updates

This policy may be updated periodically. Users will be notified of significant changes.
